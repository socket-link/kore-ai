ChatHistory

@Composable
fun ChatHistory(modifier: Modifier = Modifier, listState: LazyListState, messages: List<Chat>, isLoading: Boolean, displaySnackbar: (String) -> Unit)

A composable function that displays the chat history in a lazy column.

This function takes a list of chat messages and displays them in a scrollable column. Each message is displayed using the ChatMessage composable. If the chat is loading, a circular progress indicator is shown at the bottom of the list.

Parameters

modifier

Modifier to be applied to the LazyColumn.

listState

State of the lazy list to control scrolling.

messages

List of chat messages to be displayed.

isLoading

Boolean flag to indicate if the chat is loading.

displaySnackbar

Function to display a snackbar with a given message.
